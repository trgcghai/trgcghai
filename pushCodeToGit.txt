Cách sử dụng git/github:

-Các lệnh cơ bản:
	git init: tạo repos trong máy
	git clone: lấy từ mạng về
	git pull: đồng bộ từ mạng về máy

	git add, git add . : thêm thay đổi để chuẩn bị commit
	git commit: thêm thay đổi vào staging area
	git push: đồng bộ từ máy lên mạng

	git log, git status: check trạng thái file
	git remote: kết nối với remote repos

-Cách đẩy code lên github:
	b1: tạo 1 repos mới trên github
	b2: 'git init' để tạo repo trên máy ở folder cần tạo
	b3: 'git add' , 'git add .' để thêm tất cả thay đổi chuẩn bị commit
	b4: 'git commit -m (message)' để thêm thay đổi vào staging area
	b5: 'git remote' add origin (đường link của repos)
	b6: 'git push' -u origin master
